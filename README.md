# 📚 Online Book Borrowing Platform

A modern and responsive Online Book Borrowing Platform built with Next.js.  
This platform allows users to browse books, borrow books, manage collections, and experience a clean and interactive UI with secure authentication.

---

# 🚀 Live Demo

🔗 Live Website: online-book-borrowing-delta.vercel.app

---

# ✨ Features

- 🔐 Secure Authentication System using Better Auth
- 📚 Browse and Explore Books
- 🔍 Search and Filter Functionality
- 📖 Book Details Page
- ❤️ User-Friendly Responsive Design
- 🌙 Modern UI with Tailwind CSS & DaisyUI
- 🎬 Smooth Animations using Animate.css & AOS
- ⚡ Fast Performance with Next.js
- ☁️ MongoDB Database Integration
- 📱 Fully Responsive for Mobile, Tablet & Desktop
